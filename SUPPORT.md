# Support / 支持

## Where to ask / 在哪里提问

Use GitHub issues for:

- installation problems;
- documentation questions;
- reproducible bugs;
- release questions;
- contribution questions.

以下问题请使用 GitHub issues：

- 安装问题；
- 文档问题；
- 可复现 bug；
- 发布相关问题；
- 贡献相关问题。

## Before opening an issue / 提 issue 前

Please include:

- your operating system;
- Node and npm versions;
- the command you ran;
- whether your npm registry resolves to `https://registry.npmjs.org/` when debugging install issues;
- relevant output with secrets removed.

请提供：

- 操作系统；
- Node 和 npm 版本；
- 执行过的命令；
- 排查安装问题时 npm registry 是否解析到 `https://registry.npmjs.org/`；
- 已移除敏感信息的相关输出。

## Not support channels / 非支持渠道

Do not use public issues to share npm tokens, private repository URLs, personal data, or organization-internal details.

不要在公开 issue 中分享 npm token、私有仓库 URL、个人数据或组织内部细节。
