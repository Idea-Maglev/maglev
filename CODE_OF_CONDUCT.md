# Code of Conduct / 行为准则

## Our pledge / 我们的承诺

We want Maglev to be a respectful, evidence-driven, and constructive open-source space.

我们希望 Maglev 成为尊重他人、事实驱动、建设性协作的开源空间。

## Expected behavior / 鼓励的行为

- Be respectful and specific.
- Critique ideas and artifacts, not people.
- Provide evidence when reporting problems.
- Protect sensitive information.
- Help newcomers understand context.

- 保持尊重和具体。
- 评价观点与产物，不攻击个人。
- 报告问题时尽量提供证据。
- 保护敏感信息。
- 帮助新参与者理解上下文。

## Unacceptable behavior / 不接受的行为

- Harassment, insults, or discriminatory language.
- Publishing private credentials, personal data, or private URLs.
- Repeated off-topic disruption.
- Misrepresenting unverified claims as release facts.

- 骚扰、辱骂或歧视性表达。
- 发布私有凭据、个人数据或私有 URL。
- 反复进行无关干扰。
- 将未验证声明伪装成发布事实。

## Enforcement / 执行

Maintainers may edit, hide, close, or remove issues, comments, pull requests, or contributions that violate this code.

维护者可以编辑、隐藏、关闭或移除违反本准则的 issue、评论、PR 或贡献。
